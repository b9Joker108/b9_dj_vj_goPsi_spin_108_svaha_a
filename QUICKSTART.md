# Quick Start Guide

Welcome! This guide will help you quickly get started with FOSS video production and VJ tools on GNU/Linux.

## üöÄ 5-Minute Quick Start

### For Video Editors (Beginners)

1. **Install Kdenlive**
   ```bash
   sudo apt install kdenlive  # Ubuntu/Debian
   sudo dnf install kdenlive  # Fedora
   sudo pacman -S kdenlive    # Arch
   ```

2. **Get Sample Footage**
   - Visit [Pexels Videos](https://www.pexels.com/videos/)
   - Download 2-3 short clips

3. **Create First Project**
   - Launch Kdenlive
   - File ‚Üí New Project
   - Drag clips to timeline
   - Add simple cut/transition
   - Project ‚Üí Render
   - Done!

### For VJ Performers (Beginners)

1. **Install OBS Studio**
   ```bash
   sudo apt install obs-studio  # Ubuntu/Debian
   sudo dnf install obs-studio  # Fedora
   sudo pacman -S obs-studio    # Arch
   ```

2. **Basic VJ Setup**
   - Launch OBS Studio
   - Add video sources (Scene 1, Scene 2, etc.)
   - Set up hotkeys for scene switching
   - Start practicing transitions
   - Add effects from filters

3. **Practice**
   - Play music
   - Switch scenes to the beat
   - Experiment with transitions
   - Add color corrections/filters
