# Glossary of Terms

## Video Production Terms

### A

- **Aspect Ratio**: The proportional relationship between width and height of a video frame (e.g., 16:9, 4:3)
- **Audio Reactive**: Visuals that respond to audio input in real-time
- **Alpha Channel**: Transparency information in an image or video

### B

- **B-Roll**: Supplemental footage used to support the main narrative
- **Bitrate**: Amount of data processed per unit of time in video/audio
- **Blend Mode**: Method of combining layers (e.g., multiply, screen, overlay)

### C

- **Chroma Key**: Technique of removing a color (usually green or blue) to composite layers
- **Codec**: Compression/decompression algorithm for video/audio
- **Compositing**: Combining visual elements from separate sources into single images
- **CRF (Constant Rate Factor)**: Quality-based encoding method in x264/x265
- **Color Grading**: Process of altering and enhancing color in video
- **Cut**: Instantaneous transition between two shots

### D

- **DAW (Digital Audio Workstation)**: Software for recording and editing audio
- **Dissolve**: Transition where one shot fades out while another fades in
- **DMX**: Digital lighting control protocol
- **DNxHD/DNxHR**: Professional intermediate codecs by Avid

### E

- **EDL (Edit Decision List)**: File containing editing decisions
- **EEVEE**: Real-time render engine in Blender
- **Export**: Process of rendering final video file

### F

- **FPS (Frames Per Second)**: Number of individual frames displayed per second
- **Frame**: Single still image in a video sequence
- **FOSS (Free and Open Source Software)**: Software with source code freely available

### G

- **GPU (Graphics Processing Unit)**: Hardware for accelerating graphics rendering
- **Green Screen**: See Chroma Key

### H

- **H.264/H.265**: Common video compression standards
- **HDR (High Dynamic Range)**: Extended range of luminance

### I

- **Interlaced**: Video where each frame contains two fields (even and odd lines)
- **ISF (Interactive Shader Format)**: Shader format for real-time graphics

### J

- **JACK (Jack Audio Connection Kit)**: Professional audio server for low-latency connections
- **Jump Cut**: Cut between sequential shots of same subject

### K

- **Keyframe**: Frame that defines starting or ending point of transition/animation
- **Keying**: Process of isolating subjects based on color or luminance

### L

- **Letterbox**: Black bars above and below image to maintain aspect ratio
- **LUT (Look-Up Table)**: Preset color transformation
- **Luminance**: Brightness component of video signal

### M

- **Matte**: Mask used to define transparent areas
- **MIDI (Musical Instrument Digital Interface)**: Protocol for connecting electronic instruments
- **MLT (Media Lovin' Toolkit)**: Multimedia framework used by several editors
- **Motion Tracking**: Following movement of objects through footage

### N

- **NLE (Non-Linear Editor)**: Video editing system allowing access to any frame
- **Node**: Individual processing element in node-based workflow

### O

- **OSC (Open Sound Control)**: Protocol for communication between multimedia devices
- **Opacity**: Degree of transparency
- **Overlay**: Blend mode or layer placed over base layer

### P

- **Proxy**: Lower resolution version of footage for editing performance
- **ProRes**: Professional intermediate codec by Apple

### R

- **Render**: Process of generating final video from timeline
- **Resolution**: Number of pixels in video frame (e.g., 1920x1080)
- **RGB**: Red, Green, Blue color model
- **Rotoscoping**: Frame-by-frame tracing or masking

### S

- **Shader**: Program that runs on GPU for rendering effects
- **SOTA (State-of-the-Art)**: Most advanced/current technology
- **Sample Rate**: Number of audio samples per second (e.g., 48kHz)
- **Saturation**: Intensity/purity of color

### T

- **Timeline**: Interface showing sequence of clips in time
- **Timecode**: Method of identifying specific frames
- **Transition**: Effect between two clips

### V

- **VFX (Visual Effects)**: Computer-generated imagery added to footage
- **VJ (Video Jockey)**: Person who creates/mixes video in real-time
- **VST (Virtual Studio Technology)**: Plugin format for audio effects

### W

- **Waveform**: Visual representation of audio or video signal
- **White Balance**: Adjustment to make colors appear natural under different lighting

### Y

- **YUV**: Color encoding system (luminance + chrominance)

## VJ and Performance Terms

- **Audio-Visual (AV)**: Combined audio and visual performance
- **BPM (Beats Per Minute)**: Tempo measurement for syncing visuals to music
- **Clip**: Short video segment in VJ library
- **Crossfade**: Smooth transition between two sources
- **DMX**: Lighting control protocol often used with VJ setups
- **Generative**: Visuals created algorithmically in real-time
- **Live Coding**: Creating/modifying code in real-time performance
- **Loop**: Video that plays continuously without visible seam
- **Mapping**: See Projection Mapping
- **One-Shot**: Single-play visual element (not looping)
- **Projection Mapping**: Projecting video onto 3D surfaces
- **Syphon/Spout**: Protocols for sharing video between applications
- **VJing**: Act of live video mixing and manipulation
- **Visual Library**: Collection of clips organized for VJ performance

## Technical Abbreviations

- **ACES (Academy Color Encoding System)**: Color management framework
- **API (Application Programming Interface)**: Interface for software interaction
- **CLI (Command Line Interface)**: Text-based interface
- **CUDA**: NVIDIA's parallel computing platform
- **DCP (Digital Cinema Package)**: Format for digital cinema distribution
- **DKMS (Dynamic Kernel Module Support)**: Framework for kernel modules
- **FFmpeg**: Complete multimedia framework
- **FLOSS**: Free/Libre and Open Source Software
- **FOSS**: Free and Open Source Software
- **GEM (Graphics Environment for Multimedia)**: Pd video extension
- **GIMP (GNU Image Manipulation Program)**: Image editor
- **GLSL (OpenGL Shading Language)**: Shader programming language
- **GNU (GNU's Not Unix)**: Free software operating system
- **GPU**: Graphics Processing Unit
- **GUI (Graphical User Interface)**: Visual interface
- **LTS (Long Term Support)**: Version with extended support period
- **OCIO (OpenColorIO)**: Color management library
- **OpenCL**: Framework for parallel programming
- **OpenFX**: Plugin standard for VFX software
- **OpenGL**: Cross-platform graphics API
- **OSS (Open Source Software)**: Software with accessible source code
- **PBR (Physically Based Rendering)**: Realistic rendering technique
- **Pd (Pure Data)**: Visual programming environment
- **PPA (Personal Package Archive)**: Ubuntu software repository
- **RAM (Random Access Memory)**: Computer memory
- **SSD (Solid State Drive)**: Fast storage device
- **TTY (Teletypewriter)**: Text terminal interface
- **UI (User Interface)**: How users interact with software
- **VSE (Video Sequence Editor)**: Blender's video editor
- **WebGL**: JavaScript API for web graphics

## File Formats

### Video Formats
- **MP4**: Common container format
- **MOV**: QuickTime container format
- **MKV (Matroska)**: Open container format
- **AVI**: Legacy container format
- **WebM**: Web-optimized format

### Video Codecs
- **H.264/AVC**: Common compression codec
- **H.265/HEVC**: Newer, more efficient codec
- **VP9**: Open source codec by Google
- **AV1**: Next-gen open source codec
- **ProRes**: Professional intermediate codec
- **DNxHD/HR**: Avid professional codec

### Audio Formats
- **WAV**: Uncompressed audio
- **FLAC**: Lossless compression
- **MP3**: Lossy compression
- **AAC**: Advanced lossy compression
- **OGG**: Open source audio format

### Image Formats
- **PNG**: Lossless with transparency
- **JPEG/JPG**: Lossy compression
- **TIFF**: High quality, lossless
- **EXR**: High dynamic range format
- **SVG**: Vector graphics format

## Software Categories

### NLE (Non-Linear Editors)
Software for video editing with timeline-based interface
- Examples: Kdenlive, DaVinci Resolve, Shotcut

### Compositor
Software for combining visual elements
- Examples: Natron, Blender Compositor, Fusion

### VJ Software
Real-time video mixing and manipulation
- Examples: Veejay, FreeJ, OBS Studio

### 3D Software
Tools for 3D modeling, animation, and rendering
- Examples: Blender

### DAW (Digital Audio Workstation)
Audio recording and editing software
- Examples: Ardour, Audacity, Reaper

### Color Grading
Specialized color correction and grading tools
- Examples: DaVinci Resolve Color Page

---

## Workflow Concepts

- **Proxy Workflow**: Editing with low-res files, rendering with high-res
- **Offline/Online Editing**: Rough cut with proxies, finish with originals
- **Round-Trip**: Moving project between different applications
- **Conform**: Matching edited sequence to higher quality source
- **Render Farm**: Multiple computers rendering simultaneously
- **Batch Processing**: Automated processing of multiple files
- **Pipeline**: Complete workflow from capture to delivery

## Quality Metrics

- **Bitrate**: Mbps (megabits per second)
- **Resolution**: Pixels (1920x1080, 3840x2160)
- **Frame Rate**: FPS (24, 25, 30, 60)
- **Sample Rate**: Hz (44100Hz, 48000Hz)
- **Bit Depth**: bits (8-bit, 10-bit, 16-bit)
- **Color Space**: (sRGB, Rec.709, Rec.2020, ACES)
